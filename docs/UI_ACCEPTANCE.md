## 2026-06-27 本地浏览器验收

环境：

- 后端：`http://127.0.0.1:8001`
- 前端：`http://127.0.0.1:5174`
- 启动方式：`quant.ps1 start`
- 数据：离线 demo + 本地缓存，股票池 5199 只，缓存 16 只，K 线 11710 根。

页面巡检：

- 系统总览：正常显示 API 在线、数据准备度 100%、策略库、指标库、研究资产状态。
- 数据平台：正常显示股票池 5199、本地缓存 16、指标目录 20、缓存表格。
- 交易运行：只读展示实盘配置和人工启动命令，不提供 Web 自动下单入口。
- 研究资产：正常显示 30 条回测实验、参数网格入口、对比和导出入口。
- 策略管理：正常显示 5 个组合策略，包含 4 个由现有基础策略迁移的内置策略和示例策略；条件编辑器、保存、添加条件、添加条件组入口可见。
- 因子研究：正常显示 5 个内置因子和自定义因子入口。
- 风险控制：正常显示默认风控规则和规则评估入口。
- 回测研究：默认策略来自策略库 `组合 · MA 均线交叉`。
- 智能选股：默认策略来自策略库 `MA 均线交叉`，组合条件编辑器可见。
- AI 研究员：页面正常渲染；模型未配置时不阻塞其他模块。

交互验收：

- 回测研究：点击“运行回测”后生成绩效指标、资金曲线和 17 条交易记录，无浏览器控制台错误。
