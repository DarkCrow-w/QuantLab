# 抄底策略



### 核心指标： RSI, KDJ, VOL-TDX, BBI

**执行策略：**

+ 买入条件：当符合选股条件时，开始买入建仓。
+ 设置止损：买入当日K线最低点向下3%-5%。如果距离上一个N型结构前低较近则设为N型结构低点。
+ 止盈放飞：当股价上穿BBI线两日，并且是连续两根中大阳线（根据具体股票涨幅判断：比如波动较小的银行类，保险类股票2-4%就算中大阳线，弹性较大的科技类股票4%-7%算中大阳线），放飞一半。
+ 清仓收队：股价上涨止盈过后，连续两天跌破BBI线，清仓收队。



#### 盘面分析

##### N型结构

+ N型上涨：低点不断抬高，高点不断抬高。
+ 区间震荡：股价在一个区间内上下波动，没有明确高点低点。
+ N型下跌：低点不断降低，高点不断降低。

放量：

+ 一般放量：后一日量比前一日稍微高点。
+ 倍量：两倍（有资金入场）
+ 天量：倍量再倍量（短期大概率见顶）



### 策略选股：

+ 只看日线，选前期有放量上涨，并且之后伴随缩量回调的N形结构上涨的图形。
+ 成交量条件：20日内出现天量红柱或翻倍再翻倍红柱，并且期间不能出现高位放量大阴线。
+ 回调条件：回调到kdj的j<10或者rsi参数设置为(3,3,3)，rsi<20。
+ 缩量十字星k线（主板的：振幅4%以内，涨跌幅2%以内。创业科创：振幅7%以内，涨跌幅4%以内。）
+ MACD白线在0轴之上。
+ 股价位置：当日收盘价在BBI的±5%区间内。
